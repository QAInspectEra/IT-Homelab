# IT-Homelab

Overview

This repository documents my personal IT home lab, built to develop practical skills in Windows Server administration, Active Directory, networking, troubleshooting, and infrastructure management.

The lab environment is designed to simulate common enterprise scenarios and provide hands-on experience with system administration tasks.
