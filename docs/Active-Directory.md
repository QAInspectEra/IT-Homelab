# Active Directory

## Task

Create and configure an Active Directory test domain to practice user, group, and permission management in a Windows Server environment.

---

## What I Did

### Domain Setup

- Installed Active Directory Domain Services (AD DS)
- Promoted Windows Server 2022 to a Domain Controller
- Created a test domain
- Verified domain functionality

### Organizational Units (OU)

- Created Organizational Units (OU) for different departments
- Structured users according to organizational roles
- Tested administration through OU hierarchy

### User Management

- Created user accounts
- Configured passwords and account settings
- Tested user logon functionality

### Group Management

- Created security groups
- Added users to groups
- Used groups to simplify permission management

### Permission Management

- Assigned permissions through group membership
- Tested access control scenarios
- Verified user access based on assigned rights

---

## Result

Successfully deployed a functional Active Directory test environment.

The environment allows practical experience with:

- Domain administration
- User management
- Group management
- Organizational Units (OU)
- Permission assignment
- Access control

The setup simulates common administrative tasks performed in enterprise Windows environments.
