# DHCP

## Task

Configure and test Dynamic Host Configuration Protocol (DHCP) services within the Active Directory lab environment.



## What I Did

### DHCP Server Configuration

- Installed and configured DHCP on Windows Server 2022
- Authorized the DHCP server in Active Directory
- Verified DHCP service operation

### DHCP Scope Configuration

- Created a DHCP scope
- Configured IP address range
- Defined subnet mask and lease duration

### DHCP Options

- Configured default gateway option
- Configured DNS server option
- Configured DNS domain name option

### Client Testing

- Obtained IP addresses automatically from DHCP
- Verified correct network configuration
- Tested client connectivity

### Administration and Troubleshooting

- Reviewed DHCP leases
- Monitored address assignments
- Verified proper DHCP functionality



## Result

Successfully configured and tested DHCP services within the lab environment.

The exercises provided practical experience with:

- DHCP server administration
- Scope configuration
- IP address management
- Client network configuration
- DHCP troubleshooting

These tasks helped develop practical skills in automated network configuration and Windows Server administration.
