# Networking

## Task

Configure and test basic network communication in a virtual lab environment.

---

## What I Did

### IPv4 Configuration

- Configured network settings for Windows Server and client systems
- Assigned IP addresses
- Verified network communication between devices

### DNS Configuration

- Configured DNS settings
- Verified name resolution
- Tested communication between client and server

### Connectivity Testing

- Used Ping to verify network connectivity
- Tested communication between virtual machines
- Confirmed successful packet transmission

### Name Resolution Testing

- Used NSLookup to test DNS functionality
- Verified hostname resolution
- Identified and corrected configuration issues when necessary

### Network Diagnostics

- Investigated connectivity problems
- Verified IP configuration
- Checked DNS settings
- Performed basic troubleshooting procedures


## Result

Successfully configured and tested network communication within the lab environment.

The networking exercises provided practical experience with:
- Static IPv4 addressing
- DNS configuration and validation
- Network connectivity testing using Ping
- DNS name resolution using NSLookup
- Basic network troubleshooting and diagnostics

These tasks helped build a solid foundation for system administration and infrastructure support.
