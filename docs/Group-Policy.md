# Group Policy

## Task

Create and test Group Policy Objects (GPO) to manage user and computer settings in an Active Directory environment.

## What I Did

### GPO Creation

- Created Group Policy Objects (GPO)
- Configured policy settings
- Linked GPOs to Organizational Units (OU)

### Password Policy

- Configured password complexity requirements
- Defined minimum password length
- Configured password expiration settings

### OU Assignment

- Applied policies to specific Organizational Units
- Verified policy inheritance
- Tested policy scope and targeting

### Testing

- Updated Group Policy on client systems
- Verified policy application
- Confirmed expected configuration changes

## Result

Successfully implemented and tested Group Policy management within the lab environment.

The exercises provided practical experience with:

- Group Policy creation
- Password policy management
- Organizational Unit targeting
- Policy deployment
- Policy testing and verification

These tasks helped develop practical skills in centralized Windows administration.
