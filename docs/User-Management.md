# User Management

## Task

Create and manage user accounts and groups within the Active Directory lab environment.



## What I Did

### User Accounts

- Created user accounts in Active Directory
- Configured account properties
- Assigned passwords according to policy requirements

### Organizational Units

- Placed users into appropriate Organizational Units (OU)
- Maintained logical user structure
- Verified user organization within the domain

### Group Membership

- Added users to security groups
- Verified group membership assignments
- Tested access based on group membership

### Account Testing

- Tested user logon functionality
- Verified domain authentication
- Confirmed access permissions

### Administration

- Modified user account settings
- Reset passwords when required
- Managed account status and permissions



## Result

Successfully managed user accounts and group memberships within the lab environment.

The exercises provided practical experience with:

- User account administration
- Organizational Unit management
- Security group administration
- Domain authentication
- Permission management

These tasks helped develop practical Active Directory administration skills.
